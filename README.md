# Freelance Finance Assistant

An AI-powered chatbot for freelancers to ask about taxes, deductions, budgeting, and retirement.

## Setup
1. Create .env with GEMINI_API_KEY
2. pip install -r requirements.txt
3. streamlit run app.py

## Evaluation
20 test questions scored (see evaluation_scores.csv)
